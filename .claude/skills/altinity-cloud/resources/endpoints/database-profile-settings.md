# DatabaseProfileSettings

DELETE /setting/{id} — Removes a setting modification from a given setting profile
GET /profile/{profile}/ch-options — Lists all available database profile settings
GET /profile/{profile}/settings — Lists all modified settings for a given setting profile
POST /profile/{profile}/settings — Creates a setting modification for a given setting profile [name, value]
POST /setting/{id} — Modifies a setting for a given profile [name, value]
