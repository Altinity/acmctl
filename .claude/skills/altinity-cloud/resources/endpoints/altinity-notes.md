# AltinityNotes

DELETE /altinity-note/{id} — Removes an Altinity note
GET /environment/{environment}/altinity-notes — Lists Altinity notes for an environment
POST /altinity-note/{id} — Updates an existing Altinity note [severity, title, message]
POST /environment/{environment}/altinity-notes — Creates a new Altinity note for an environment [severity, title, message]
